# Contributing

We'd love your help!

## How to Contribute

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Test in After Effects
4. Commit with clear messages
5. Push and open a Pull Request

## Bug Reports

Please include:
- AE version
- OS (Windows/Mac)
- Steps to reproduce
- Screenshots if helpful

## Feature Requests

Describe what you'd like and why it would help!
