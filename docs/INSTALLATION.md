# Installation Guide

## Windows

1. Download `Auto Beat Marker.jsx`
2. Navigate to: `C:\Program Files\Adobe\Adobe After Effects [VERSION]\Support Files\Scripts\ScriptUI Panels`
3. Paste the file
4. Restart After Effects
5. Window > Auto Beat Marker Pro

## Mac

1. Download `Auto Beat Marker.jsx`
2. Navigate to: `/Applications/Adobe After Effects [VERSION]/Scripts/ScriptUI Panels`
3. Paste the file
4. Restart After Effects
5. Window > Auto Beat Marker Pro

## Verify Installation

- After Effects loads without errors
- Script appears in Window menu
- UI displays properly
