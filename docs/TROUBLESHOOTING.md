# Troubleshooting

| Problem | Solution |
|---------|----------|
| "No audio amplitude layers found" | Click "Convert Audio" first |
| "No peaks found" | Lower sensitivity or reduce smoothing |
| Too many markers | Lower sensitivity or increase gap |
| Too few markers | Raise sensitivity or decrease gap |
| Wrong BPM | Manually enter correct BPM |
| Script won't open | Reinstall to correct Scripts folder |

## Can't find Scripts folder?

**Windows**: Search for "Adobe After Effects" folder
**Mac**: Applications folder → right-click AE → Show Package Contents
